Detection of Layer 7 DDoS Attacks Using NLP

Complete Step-by-Step Guide for Beginners

Step 1: Understand the Problem Statement

Layer 7 DDoS attacks target the application layer (HTTP/HTTPS), overloading servers with legitimate-looking traffic. The goal of this project is to detect such patterns using NLP on web server logs to classify behavior as normal or malicious.

Step 2: Set Up the Project Environment

Requirements: - Python 3.9+ - VS Code or Jupyter Notebook - Libraries: pandas, numpy, matplotlib, seaborn, scikit-learn, nltk, streamlit Recommended structure: project-root/ ├── data/ ├── notebooks/ ├── app/ ├── requirements.txt └── README.md

Step 3: Collect or Simulate Web Logs

Use real access logs or simulate data using tools like: - Apache Log Generator - Python scripts with timestamps and GET/POST requests Sample Log Format: 192.168.1.10 - - [06/Apr/2025:12:01:22 +0000] "GET /login HTTP/1.1" 200 214

Step 4: Load and Parse Logs

Use Python's `re` module or `apache_log_parser` to extract features. Regex Example: pattern = r'(?P\\d+.\\d+.\\d+.\\d+) - - \\[(?P.*?)\\] "(?P\\w+) (?P\\S+) HTTP/\\d.\\d" (?P\\d{{3}}) (?P\\d+)' Use pandas to store it in DataFrame for further analysis.

Step 5: Preprocess the Data

- Convert timestamp to datetime - Normalize all URL paths (e.g., /Login → /login) - Extract temporal features: hour, day, minute - Clean status codes and filter bots or irrelevant requests Example: df['timestamp'] = pd.to_datetime(df['timestamp'], format='%d/%b/%Y:%H:%M:%S %z')

Step 6: Feature Engineering

- Count requests per IP in time window - Calculate average response time - Use NLP on URLs and user-agents (TF-IDF, Word2Vec) - Detect repetitive patterns in request headers or payloads

Step 7: Apply NLP Techniques

- Tokenize URL paths and use n-grams - Vectorize with CountVectorizer or TfidfVectorizer - Perform clustering or frequency analysis on user-agents

Step 8: Train Detection Models

- Label data (manually or with threshold rules) - Train machine learning models like: - Random Forest - Logistic Regression - Isolation Forest (unsupervised) Use scikit-learn to split data, train and evaluate: from sklearn.ensemble import RandomForestClassifier model = RandomForestClassifier() model.fit(X_train, y_train)

Step 9: Visualize & Evaluate Results

- Use confusion matrix, accuracy, precision-recall - Plot request patterns using matplotlib/seaborn - Visualize abnormal spikes by IP and endpoint

Step 10: Deploy as a Web App

Use Streamlit or Flask to create a simple interface where: - Logs can be uploaded - Anomalies are detected and visualized - Real-time alerts are shown for suspicious IPs
